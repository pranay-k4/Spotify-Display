# Spotify-Display
An interactive device that displays the currently playing Spotify track and allows basic playback control using physical buttons.

## Features
- Displays current Spotify track and artist
- Physical buttons for playback control
- WiFi connectivity
- Compact desk device

## Hardware
- 1.8 inch SPI TFT LCD Display Module
- ESP32 C3 Mini
- Tactile Switches
